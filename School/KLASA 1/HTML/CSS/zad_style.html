Jakie zadanie spełnia arkusz CSS?

CSS to język kodowania używany do stylizowania i formatowania stron internetowych

Na czym polega kaskadowość stylów?

Kaskadowość w CSS oznacza, że wiele reguł stylów może dotyczyć tego samego elementu HTML

Jakie są sposoby wstawiania stylów?

CSS może być osadzony jako arkusz zewnętrzny, zagnieżdżony oraz 'inline'

Jaka jest hierarchia ważności stylów?

Style o wyższym priorytecie ważności (na początku listy) mają pierwszeństwo w modyfikowaniu elementów dokumentu. Style niżej w hierarchii mogą zmienić formatowanie jakiegoś elementu, tylko jeśli style o wyższym priorytecie nie dotyczą tego konkretnego elementu.

Ile stylów można umieszczać jednocześnie w dokumencie HTML?

W dokumencie html można dołączyć dowolną liczbę stylów
